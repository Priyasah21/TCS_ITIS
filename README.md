# TCS ITIS Preparation Hub 🚀

Welcome to the **TCS ITIS Preparation Hub**!

This repository is created to help students prepare for the **TCS IT Infrastructure Services (ITIS)** assessment and interview process. All the resources shared here are organized topic-wise to make your preparation structured and efficient.

## 📚 Topics Covered

* Business Skills (BizSkills)
* UNIX & Linux
* User Interface Fundamentals
* Windows 10 & Windows Server
* PowerShell
* Networking
* ITIL Basics
* Computer Hardware
* Cloud Computing
* Databases
* Virtualization
* Office 365
* Cyber Security
* Scenario-Based MCQs
* Interview Preparation
* Practice Questions & Notes

## 📂 Repository Structure

Each folder contains:

* Study Notes
* Important Concepts
* MCQs
* Revision Material
* Additional Resources (where applicable)

## 🎯 Who Is This For?

* Students preparing for the TCS ITIS assessment
* Candidates waiting for onboarding and looking to upskill
* Anyone interested in learning IT Infrastructure concepts

## 💡 How to Use This Repository

1. Start with the topic you are least familiar with.
2. Go through the notes thoroughly.
3. Practice the MCQs after completing each module.
4. Revise regularly and make your own short notes.
5. Apply concepts through hands-on practice whenever possible.
