# Quick Revision

### Wildcards

* `*` → Zero or more characters
* `?` → Exactly one character
* `[abc]` → Match one character from set

### Redirection

* `>` → Overwrite output
* `>>` → Append output
* `<` → Input redirection

### Search Commands

* `grep` → Search text patterns
* `find` → Search files/directories
* `locate` → Fast file search
* `which` → Executable location
* `whereis` → Command information

### Frequently Asked Commands

| Command | Purpose                 |
| ------- | ----------------------- |
| head    | First lines             |
| tail    | Last lines              |
| tail -f | Live monitoring         |
| wc      | Count lines/words/chars |
| grep    | Search text             |
| sort    | Sort data               |
| uniq    | Remove duplicates       |
| cut     | Extract columns         |
| tr      | Replace characters      |
