# Most Repeated Interview Questions

### What is Active Directory?

Microsoft's directory service for managing users, computers, and resources.

---

### What is a Domain Controller?

A server that provides authentication and authorization services.

---

### What is an Organizational Unit?

A logical container used to organize Active Directory objects.

---

### What is the difference between Authentication and Authorization?

Authentication verifies identity; Authorization determines permissions.

---

### Why does Active Directory depend on DNS?

DNS helps clients locate domain controllers and services.
